# IT Support Ticket Resolution – User Login Issue

## Objective
To simulate resolving a common IT support issue using structured troubleshooting and user support techniques.

---

## Scenario
A user was unable to log into their system after multiple failed attempts. The objective was to identify the issue and restore access while maintaining security best practices.

---

## Troubleshooting Process
- Confirmed the issue with the user  
- Verified username and password accuracy  
- Checked if the account was locked due to multiple failed attempts  
- Reset the user password  
- Ensured system accessibility after reset  

---

## Findings
- Account was locked due to repeated failed login attempts  
- No evidence of malicious activity detected  

---

## Resolution
- Password reset performed  
- Account unlocked  
- User successfully logged into the system  

---

## Outcome
- Issue resolved efficiently  
- User access restored  
- Minimal downtime experienced  

---

## Skills Demonstrated
- IT support and troubleshooting  
- User communication  
- Problem-solving  
- Basic account security handling
